# Output Schema

## Daily Report Schema

日报输出必须固定为以下结构：

1. 今日核心结论
2. 关键数据摘要
3. 节奏预测
4. 核心问题诊断
5. 三大问题拆解
6. 优先行动清单
7. 团队PK分析
8. 风险预警
9. 专项问题模块
10. 一句话总结
11. 最终目标达成检查

---

## Weekly Report Schema

周报输出必须固定为以下结构：

1. MTD核心经营数据总览
2. 整体趋势对比
3. 团队拆解
4. 来源拆解
5. MTD节奏判断
6. 链路倒推诊断
7. 问题团队与销售
8. 风险预警
9. 下周行动计划

---

## Action Item Schema

每条行动必须包含：

- 对象
- 动作
- 数量
- 完成时间
- 优先级（P0 / P1 / P2）
